description: Create a comprehensive Design System and UI tokens for "Kipo", a modern and next-generation Mexican E-Invoicing (CFDI 4.0) and POS SaaS tailored for freelancers and micro-businesses (2-3 employees). 
---

### 1. BRAND IDENTITY & BLURB
* Name: Kipo
* Tagline: Factura. Sin drama.
* Personality: Fast-casual, friendly-professional, trustworthy, and ultra-clean (FinTech vibe). It must balance absolute financial precision with zero bureaucratic stiffness.
* Focus: Mobile-First web app, mobile responsive landing page, and automated dashboard layouts.

### 2. VISUAL TOKENS & TYPOGRAPHY
* Primary Brand Colors: Deep Magenta/Plum (#9A1B5E to #B81D53 gradient) as the core identity, accompanied by a vibrant neon accent color for high-conversion CTAs. Use clean dark grays for backgrounds and layout structures to keep a premium aesthetic.
* Typography Pairing: 
  - Headings & Hero Titles: "Plus Jakarta Sans" (Bold/SemiBold) for a modern, geometric, and energetic impact.
  - Body Text & App Interface: "Inter" (Regular/Medium) for absolute pixel-perfection and high legibility on 6-inch mobile screens.
  - Financial Data & Tables: "Geist Mono" for monospace numbers, RFC inputs, tax calculations, and perfectly aligned invoice tables.

### 3. ILLUSTRATION & IMAGES STYLE
* Preference: Use flat vector images and clean geometric illustrations.
* Style Guide: Avoid 3D renders, skeuomorphism, or busy stock photos. Focus on minimalist, flat 2D vector shapes, thin lines, and solid or subtle gradient color blocks that match the primary palette. Illustrations should represent abstract metaphors of financial control, speed, and cloud automation.

### 4. COMPONENT & CARD DESIGN REQUIREMENTS
* UI Style: Clean, flat layout with smooth rounded corners (border-radius: 12px to 16px). High contrast on interactive components.
* Core Elements Needed:
  - Pay-as-you-go checkout micro-cards ($3.50 MXN per stamp).
  - "Premium Personalized Invoices" previewer component (where users can customize logos, layout structures, and brand fonts).
  - Quick AI-reading data inputs (optimized for scanning Tax ID / Constancia de Situación Fiscal via QR code).
  - Mobile-first dead-simple transaction charts and bank reconciliation cards.
